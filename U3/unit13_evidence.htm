Evidence for Website Development

The task that I had to do is:
I had to create a design for an interactive website. The main focus of the 'Mobile World Station' website is to promote mobile phones, accessories and services. The website I had to make it on;

  • Target anyone who is interested in buying mobile phones.

  • Load easily and be quick and easy to navigate (consider using menus).

  • Contain at least eight linked pages.

  • Contain at least one table.

  • Have a consistent use of text, color schemes and page styles in all of the web pages.

  • Be interesting and have at least four suitable, different original and/or ready-made, embedded multimedia/digital assets, e.g. digital animation, digital graphics, digital audio, digital video.

  • Contain some interactive features, e.g. a feedback form to collect customer enquiries.

  • Have at least two external hyperlinks to other interesting webpages.

  • Be internally linked, with easy-to-navigate menus.

  • Contain a link to an online forum.

Software that's needs to be used to make the websites is called Microsoft Expression Web
